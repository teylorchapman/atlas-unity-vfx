# atlas-unity-vfx

### Summary
This is the beat detection game for the VFX and Shadergraph course at Atlas School. I used free assets for the Brick Breaker/Breakout gamebase and created a 3d noise flowfield using scripts. 


### Getting Started
- install the Unity editor version 2021.3.* or newer.
- Have a basic understanding of the C# coding language.
